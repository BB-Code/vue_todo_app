# vue_todo_app
![](https://img.shields.io/badge/vue-2.5.2-green.svg)

## 获取

```
git clone https://github.com/BB-Code/vue_todo_app.git
```

## 安装

``` bash

cd vue_todo_app

# 安装依赖包
npm install

# 服务跑在 localhost:8080
npm run dev

```
